Aprender Programação Web

JavaScript

JavaScript Básico

O JavaScript básico permite criar interatividade em páginas web, como alertas, botões e validação de formulários.

JavaScript Intermédio

No nível intermédio, aprende-se a trabalhar com funções, arrays, objetos e eventos de forma mais estruturada.

JavaScript Avançado

O JavaScript avançado envolve conceitos como promises, async/await, manipulação avançada do DOM e frameworks.

Python

Python Básico

Python básico ensina variáveis, tipos de dados, condições e ciclos, sendo ideal para iniciantes.

Python Intermédio

Neste nível, exploram-se funções, módulos, ficheiros e estruturas de dados mais complexas.

Python Avançado

Python avançado aborda programação orientada a objetos, bibliotecas avançadas e desenvolvimento de aplicações.
